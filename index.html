Sybil
Download Documentation

Sybil is 3kb minified & less than 1kb gzipped.

Sybil is not a carousel replacement but a lightweight "feature wall" component.

Sybil supports slide transitions with any CSS animation.

Sybil makes very few CSS assumptions.

Sybil supports IE 8 +, FireFox 3 +, Safari 2 + & requires jQuery 1.4 or higher.

Sybil is 100% open source & is licensed under MIT.

Sybil documentation

Basic

$('#sybil').sybil();

Sybil requires two inner div's; both with classes. However these div's may be nested inside any number of other elements.


  // Initialize Sybil on element.
  $('#sybil').sybil({
    currentSlide: '.one-div',
    nextSlide: '.two-div'
  });
